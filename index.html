7.3 循环语句

重复地执行某些相同或相似的操作

7.3.1while循环

格式：

​ 初始变量
​ while(条件表达式){
​ 循环体; // 当条件表达式为true的时候执行的语句
迭代因子;
​ }

含义：

初始变量先进行执行,当遇到while之后,准备要循环,

计算条件表达式, (true, false), true, 执行循环体, 再执行迭代因子(变化的量),

当整个循环体执行结束之后, 再次回到条件表达式处执行,再次计算条件表达式.

7.3.2 do…while 循环

最少执行循环体一次再进行判断

格式：

do{

​ 循环体；

​ 迭代因子；

}while(条件表达式);

7.3.3 for 循环

格式：

for(初始表达式;条件表达式;迭代因子){

​ 循环体；

}

3. 参数列表不同

8.8递归

8.8.1递归的基本思想
自己调用自己
一个使用递归技术的方法将会直接或者间接的调用自己
利用递归可以用简单的程序来解决一些复杂的问题
大多数排序使用的就是递归算

递归结构包括两个部分
定义递归头
什么时候不调用自身方法。如果没有头，将陷入死循环。
递归体
什么时候需要调用自身方法以及自身需要做什么操作。

递归的缺陷
简单的程序是递归的优点之一。
递归调用会占用大量的系统堆栈，内存耗用多

注意
任何可用递归解决的问题也能使用迭代解决
在不强调效率并且有足够的内存空间时，可以采用
在要求高性能的情况下尽量避免使用递归，递归调用既花时间又耗内存
